# 📘 정보처리산업기사 CBT 모의시험 시스템

정보처리산업기사 필기 기출문제를 실제 시험과 유사한 CBT(Computer Based Test) 환경에서 연습할 수 있는 웹 애플리케이션입니다.

## ✨ 주요 기능

- **실제 시험과 동일한 환경**
  - 3개 과목(정보시스템 기반 기술, 프로그래밍 언어 활용, 데이터베이스 활용) 구성
  - 각 과목당 20문항, 총 60문항
- **자동 채점 및 결과 분석**
  - 제출 즉시 과목별 점수 및 총점 자동 채점
  - 과락(과목별 40점 미만) 및 최종 합격(평균 60점 이상) 여부 자동 판정
- **효율적인 학습 지원**
  - 문제별 즉각적인 정/오답 피드백 (O/X 표시)
  - '해설 보기' 옵션을 통해 틀린 문제에 대한 상세 해설 확인
  - 실제 시험과 유사한 UI로 실전 감각 극대화

## 🗺️ 향후 계획

- **AI 기반 개인화 문제 추천**
  - 사용자가 자주 틀리는 문제 유형을 분석하여 AI가 유사한 문제를 생성 및 추천
  - 단순 기출문제가 아닌, 개인의 취약점을 보완하기 위한 새로운 문제 제공
- **오답 노트**
  - 틀린 문제만 모아서 다시 풀어볼 수 있는 기능 추가
- **학습 통계 대시보드**
  - 과목별 정답률, 전체 학습 진행도 등 시각적인 통계 데이터 제공
